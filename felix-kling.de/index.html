Projects

Current and past projects that I created:

  • astexplorer.net
    An online editor to explore abstract syntax trees of various languages and prototype plugins for popular code transforms (Babel, ESLint, …).
  • react-docgen
    A static code analyzer to extract API documentation for React components.
  • jscodeshift
    An AST based codemod (one-time code transformation) tool for JavaScript.
  • JSBasics (2017)
    An interactive introduction to the fundamentals of JavaScript.
  • jsnetworkx.org (2012 - 2015)
    A port of the Python graph library NetworkX to JavaScript (incomplete, unmaintained).
  • Game of Life (2011)
    A Game of Life implementation in JavaScript, with <canvas>.

JavaScript

JavaScript is the language I’m most familiar with and I enjoy helping people understand its oddities. I would like to build more interactive tools to help with that. To start with, JavaScript “loose” comparison step by step shows which steps of the comparison algorithm are taken for specific values.
